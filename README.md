### Installation  
There are two ways to get this repo:

#### Method 1 (Easy) WIP
Get the shellcript which will fetch the files automatically for ease.

```
  wget https://github.com/RaySlash/dotfiles/blob/main/autoconfigure.sh && sh
```
NOTE: Give proper permission to the script if it fails to load (chmod +x)

#### Method 2 (Legacy)
First clone this repository:

```
  git clone https://github.com/RaySlash/dotfiles
```

Navigate and Copy paste the config files to home dir (/home/user/)

```
  cd dotfiles && cp .zshrc ~/.zshrc   
```
